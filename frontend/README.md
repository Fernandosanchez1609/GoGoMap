## 🎨 Stack Tecnológico (Frontend)

Este proyecto utiliza las siguientes tecnologías para el desarrollo de la interfaz de usuario y la visualización de los mapas:

* **Librería Principal:** React
* **Lenguaje:** TypeScript
* **Herramienta de Construcción (Bundler):** Vite
* **Estilos:** Tailwind CSS
* **Librería de Componentes:** shadcn/ui
* **Mapas Interactivos:** React-Leaflet

### ⚙️ Requisitos previos del entorno
Para ejecutar el entorno de desarrollo local del frontend, asegúrate de tener instalado:
1. **Node.js** (versión 18 o superior recomendada).
2. Un gestor de paquetes como **npm**, **yarn** o **pnpm**.
